Plume forms

What is Plume forms?

Plume forms is an avian-centric, declarative and reactive way to rapidly build forms in Angular. It's low code nature empowers you to rapidly build the most complex forms with without writing a whole lot of code. While being a rather opinionated, it was still built with flexibility in mind; after all, not everybody likes the Angular Material design system. Having said that, the built-in form components that come with the package do implement the Angular Material design system components, but can easily be overridden by extending the base class and overriding the template.

Is Plume forms for me?

Ask yourself. Do you have to build similar looking forms over and over? Are you tired of stringing together template and typescript? Do you like birds? If the answer to either of these questions is yes, Plume forms is for you.

Installing Plume forms

Installing Plume forms is as easy as installing any other NPM dependency. Simply run the following commands in your project root. The forms rely on the utils package for a modicum of its functionality. 

$ npm install --save @plume-org/forms
$ npm install --save @plume-org/utils

Or use yarn

$ yarn add @plume-org/forms
$ yarn add @plume-org/utils
